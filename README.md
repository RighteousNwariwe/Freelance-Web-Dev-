# Righteous Nwariwe — Freelance Web Dev Portfolio

A single-page portfolio and lead-generation site for freelance web development services in South Africa.

## Features

- Services, portfolio, pricing, FAQ, and contact quote modal
- EmailJS-powered quote form (`service_Freelance` / `template_su12eb6`)
- WhatsApp fallback for direct contact
- Mobile-responsive layout with scroll animations

## Pricing Tiers

| Tier | Price | Best for |

| Micro | R1,000 | Single-page landing for small businesses |

## Local Development

Open `index.html` in a browser, or serve locally:

```bash
npx serve .
```

## Deployment

Deploy the project root to any static host (Netlify, Firebase Hosting, GitHub Pages, etc.). No build step required.

## Files

- `index.html` — Main site
- `favicon.svg` — Site icon
- `site.webmanifest` — PWA manifest
- `robots.txt` — Search engine directives
